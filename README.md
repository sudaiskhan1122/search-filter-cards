# Search Filter Cards Project

This project is a **dynamic user card interface** built using **HTML, CSS, and JavaScript**.  
It is designed for **learning purposes** to understand how to:

- Create HTML elements dynamically using `createElement`
- Manipulate the DOM
- Filter and display data in real-time
- Apply responsive and interactive UI effects using CSS

The project displays a list of users as **cards** containing their **photo, name, and bio**. Users can type in the search box to **filter cards live** based on the user’s name.

---

## Features

- **Dynamic card generation**: All cards are created using JavaScript `createElement`, making the DOM manipulation fully dynamic.
- **Live search filter**: Typing in the search box filters cards in real-time.
- **Responsive layout**: Cards wrap neatly on smaller screens.
- **Hover effect**: Cards slightly enlarge when hovered for better interactivity.
- **User-friendly design**: Clean and readable interface with gradient overlay on images.

---

## Learning Outcomes

This project helps you understand:

- How to **loop through data arrays** and create HTML elements dynamically.
- How to **manipulate the DOM** efficiently with JavaScript.
- How to **filter arrays** based on user input.
- How to **style cards** with CSS, including gradients and hover transitions.
- How to **structure a small front-end project** and separate HTML, CSS, and JS for modularity.

---

## How to Use

1. Clone or download the repository.  
2. Open `index.html` in your browser.  
3. Type a name in the search input to filter the cards dynamically.  

---

## Technologies Used

- **HTML5** for structure  
- **CSS3** for styling, layout, and hover effects  
- **JavaScript** for DOM manipulation and live search filter
