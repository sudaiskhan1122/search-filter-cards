# Search Filter Cards Project

This is a simple **search filter UI project** built with **HTML, CSS, and JavaScript**.  
It dynamically displays a list of users as cards with their **name, picture, and bio**, and allows live filtering using the search input.

## Features

- Dynamically created cards using `createElement`
- Live search filter (case-insensitive)
- Responsive card layout with hover effects
- Simple and clean UI

## How to Use

1. Open `index.html` in your browser.
2. Type in the search box to filter users by name.
3. Cards will update automatically as you type.
